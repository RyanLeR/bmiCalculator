1. Download repo files
2. Open terminal and navigate to root folder of download.
3. Make sure node is installed on your device.
4. Initialise NPM in repo root folder.
5. Using NPM, run following:
    npm i express
    npm i body-parser
6. In terminal, now run: >nodemon server.js
7. Open browser and in URL bar, enter: localhost:3000/bmicalculator
8. Fill in values and click on calculate. Result should appear in new page.
